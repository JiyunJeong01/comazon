# comazon
Prisma를 이용한 RDBMS 쇼핑몰 API 개발
<br><br><br>

## 🖥️ 레포지토리 소개
교내 연합 동아리 코드잇 부스트에서 스터디용으로 구현한 Prisma 이용 RDBMS 쇼핑몰 api
<br><br>

### ⚙️ 개발 환경
- `node.js`
- `WSL`
- **IDE** : Visual Studio Code
- **Framework** : Express.js
- **Database** : PostgreSQL
- **ORM** : Prisma
